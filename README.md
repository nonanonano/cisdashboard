# cis流デイリートレード・ダッシュボード

このリポジトリは、日本株の短期順張り戦略で知られるcis氏のトレードスタイルを参考に、
サラリーマン投資家向けのデイリーダッシュボード構築を目指すものです。

## 目的
- 一日一度のチェックで売買判断ができるダッシュボードを実装する。
- cis氏が重視する「順張り」「損小利大」「出来高重視」「地合い重視」に沿った指標を用意する。

## ダッシュボード構成
1. **市場環境パネル**
   - 主要指数のトレンド (日経平均・TOPIX・マザーズ)
   - 騰落レシオ、日経VI、為替 (USD/JPY) など
   - 地合い悪化時は新規買いを控えるなどの行動指針を示す
2. **個別銘柄パネル**
   - 各銘柄の移動平均線、出来高変化率、モメンタム指標などを表示
   - 条件を満たす銘柄に対して「買い」「見送り」「利確」「監視」「損切り」を判定

## 使い方
今後、HTML/JavaScript を用いてダッシュボードを構築予定です。
現時点では `index.html` に簡単なプレースホルダのみを置いています。
